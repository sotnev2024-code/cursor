# Инструкция по установке и запуску Python версии

## Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

Или если используете виртуальное окружение (рекомендуется):

```bash
# Создать виртуальное окружение
python -m venv venv

# Активировать виртуальное окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt
```

### 2. Запуск сервера

```bash
python app.py
```

Сервер автоматически:
- Создаст базу данных SQLite в папке `database/`
- Создаст все необходимые таблицы
- Заполнит начальными данными (категории, настройки)

### 3. Доступ к приложению

- **Фронтенд**: http://localhost:5000/
- **API**: http://localhost:5000/api

## Преимущества Python версии

1. **SQLAlchemy ORM** - удобная работа с БД через объекты
2. **Автоматическая миграция** - таблицы создаются автоматически
3. **Типизация** - можно добавить type hints для лучшей поддержки
4. **Расширяемость** - легко добавить новые функции

## Отличия от Node.js версии

- Порт по умолчанию: **5000** (вместо 3000)
- База данных создается автоматически при первом запуске
- Не нужно запускать отдельный скрипт инициализации БД

## Разработка

При запуске `python app.py` автоматически включается режим отладки (`debug=True`), который:
- Перезагружает сервер при изменении файлов
- Показывает детальные ошибки в браузере
- Включает интерактивный отладчик

## Решение проблем

### Ошибка импорта модулей

Убедитесь, что вы находитесь в корневой директории проекта и все файлы на месте.

### Ошибка подключения к БД

Убедитесь, что папка `database/` существует и доступна для записи.

### Порт уже занят

Измените порт в `app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # Используйте другой порт
```
